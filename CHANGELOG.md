# Changelog

All notable changes to this project will be documented in this file.

## v0.1.0
- Initial release with SQL-based local data exploration.
- Supported formats: Parquet, CSV/TSV, JSON/JSONL, Arrow/IPC, Excel.
- Drag-and-drop, file picker, schema preview, and virtualized results table.
- Export query results to CSV and XLSX.
- Built-in sample datasets and sheet selector for Excel.
